# Linux Tips & Tricks

These make the terminal easier and faster.

Tab autocomplete
- Type part of a file or folder name, then press Tab to complete it.

Up/Down arrows
- Press Up to repeat your last command.
- Press Down to go forward in your command history.

Clear the screen
```bash
clear
```

Where am I?
```bash
pwd
```

What files are here?
```bash
ls
```
