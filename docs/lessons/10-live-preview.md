# Live Preview (VS Code)

Live Preview shows your page and refreshes when you save.

What a “server” means
- A server is a program that shows your files in the browser at a web address like http://localhost:8000.

Install it
1) Open Extensions (left sidebar).
2) Search for “Live Preview”.
3) Install the Microsoft extension.

Use it
1) Right‑click `docs/index.html`.
2) Choose “Show Preview”.

If it doesn’t update
- Save the file.
- Make sure you opened the correct `index.html` inside `docs/`.
